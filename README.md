# HOOTSim
N-Body Simulator

## Setup
```
git clone https://github.com/jdinovi/HOOTSim.git
cd HOOTSim
export HOOTSIM_PATH=path/to/hootsim/directory  # we recommend adding this to your shell configuration script (i.e. .zshrc, .bashrc)
```

## Run The Simulator:
* `make build`
*  `./bin/HOOTSim`

## Run Test Suite:
* `make test`

## Run Code Coverages:
* `make coverage`
